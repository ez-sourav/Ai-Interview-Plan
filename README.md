# 🚀 AI Interview Planner

Ace your next interview with **AI-powered preparation**.  
AI Interview Planner is a smart web application that helps users prepare for interviews through personalized plans, AI-generated questions, and structured learning roadmaps.

🌐 **Live Demo:** https://interviewplanner.vercel.app/

---

## ✨ Overview

AI Interview Planner is built to simplify interview preparation by combining **modern web technologies** with **AI-driven insights**.  
It allows users to prepare smarter, identify weak areas, and practice effectively for their target roles.

---

## 🧠 Key Features

- 🎯 **Personalized Interview Plans** based on selected roles  
- 🤖 **AI-Generated Questions** (Technical + Behavioral) using Gemini API  
- 📊 **Skill Gap Analysis** to identify weak areas  
- 🛣️ **Structured Learning Roadmap** for improvement  
- 🔐 **Authentication System** (Login / Register / Logout)  
- ⚡ **Fast & Responsive UI** for smooth user experience  
- 📱 **Fully Responsive Design** (Mobile + Desktop)
